# Prompt Examples

Use these prompts with Codex.

## Build a new page

Use the `frontend-production-shadcn` skill.

Build a production-quality React + Tailwind + shadcn/ui page for [PRODUCT CONTEXT].
Do not make it look like an AI demo.
First provide the design plan, then code.
Must cover hover, loading, empty, error, disabled, focus-visible, mobile, and overflow states.

## Redesign bad UI

Use the `frontend-production-shadcn` skill.

This page feels cheap/demo-like. First diagnose why, then redesign it as a production product UI.
Preserve the product intent, improve information hierarchy, reduce decoration, add missing states, and implement responsive behavior.

## Force self-review

Use the `frontend-production-shadcn` skill.

Before writing code, list 5 ways your initial design could look cheap and fix them.
Then output the corrected implementation.
